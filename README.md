# sao-hypernova-react

> Hypernova with React boilerplate

## Usage

Install [SAO](https://github.com/saojs/sao) first.

```bash
yarn global add sao
# or
npm i -g sao
```

### From git

```bash
sao ara-framework/create-hypernova-react my-project
```
